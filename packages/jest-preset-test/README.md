# @jameslnewell/jest-preset-test

A test preset for my personal projects that use `jest`.

## Installation

```
yarn add --dev @babel/core @jameslnewell/jest-preset-test
```

## Usage

`jest.config.js`

```
{
  projects: [
    displayName: 'test',
    preset: '@jameslnewell/jest-preset-test'
  ]
}
```
