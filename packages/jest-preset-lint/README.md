# @jameslnewell/jest-preset-lint

A lint preset for my personal projects that use `jest`.

## Installation

```
yarn add --dev @jameslnewell/jest-preset-lint
```

## Usage

`jest.config.js`

```js
module.exports = {
  projects: [
    displayName: 'type',
    preset: '@jameslnewell/jest-preset-lint'
  ]
}
```
